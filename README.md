# Lumina Store - E-commerce Application

A modern e-commerce web application built with Next.js, featuring a fully functional shopping cart, product filtering, and responsive design.

## Features

- Browse and search products in real-time
- Filter by category and price range
- Sort products by price and rating
- Add/remove items from shopping cart
- Select products for checkout with total calculation
- Responsive design for mobile and desktop
- Contact form for customer inquiries
- User account page with demo profile

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Radix UI Components
- FakeStore API for product data

## Getting Started

1. Clone the repository
2. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`
3. Run the development server:
   \`\`\`bash
   npm run dev
   \`\`\`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

\`\`\`bash
npm run build
npm start
\`\`\`

## Project Structure

- `/app` - Next.js pages and layouts
- `/components` - Reusable React components
- `/lib` - Utilities, services, and context providers
- `/public` - Static assets
